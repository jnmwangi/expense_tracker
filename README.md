# Expense Tracker
This is a flask application for tracking expenses, expecially the ones done by M-Pesa and Mobile Apps

## Features
- User will be able to create custom categories or use existing ones
- User will be able to add the expenses

## Advanced features
- Auto populate the expenses from SMS received from M-Pesa and Mobile Apps
- Integrate AI for insights and prediction

## Technologies
- Flask 
- Postgresql
- Deploy in Google Cloud
- React

## Sample .env
```
    FLASK_APP=
    FLASK_RUN_PORT=5020
    FLASK_DEBUG=True
    FLASK_SQLALCHEMY_DATABASE_URI = postgresql://postgres:postgres@localhost:5433/expenses_db

    REACT_BASE_URL=
```
